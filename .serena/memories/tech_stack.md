# White Room Tech Stack

## Programming Languages

- **TypeScript 5.3+**: Authoritative SDK implementation
- **C++17**: JUCE audio engine
- **Swift 5.9+**: Native Apple platforms
- **Python**: Tooling and scripts

## Core Frameworks & Libraries

### SDK (TypeScript)
- **Node.js 18+**: Runtime
- **Vitest 3.2.4**: Testing framework
- **ESLint 8.57**: Linting
- **Prettier 3.1**: Formatting
- **Turbo 2.5**: Build system
- **TypeScript 5.3**: Type checking

### JUCE Backend (C++)
- **JUCE**: Audio framework
- **CMake 3.25+**: Build system
- **CTest**: Testing

### Swift Frontend
- **SwiftUI**: UI framework
- **Swift Package Manager**: Dependency management
- **Xcode 15+**: IDE (for Apple platforms)
- **swift-format**: Code formatting

## Build Systems

- **TypeScript**: npm workspaces + turbo
- **C++**: CMake
- **Swift**: Swift Package Manager
- **Xcode projects**: Generated for iOS/tvOS apps

## Development Tools

- **Beads (bd)**: Mandatory task tracking
- **SpecKit**: Specification-driven development
- **Confucius**: Hierarchical memory system
- **Git**: Version control
- **Husky**: Git hooks

## Platform Requirements

### Minimum Versions
- **Swift**: 5.9+
- **Node.js**: 18.0.0+
- **npm**: 9.0.0+
- **CMake**: 3.25+
- **Xcode**: 15+
- **macOS**: 14+ (for development)

### Target Platforms
- **tvOS**: 17+ (MVP)
- **iOS**: 17+
- **macOS**: 14+
