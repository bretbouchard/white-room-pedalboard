## Composition Tools

### Timeline-Based Sequencing

**Linear Timeline**:
- Intuitive timeline editor for arranging music
- Drag-and-drop regions for easy composition
- Visual representation of song structure
- Support for multiple time signatures
- Tempo and key signature changes

**Project Management**:
- Multiple projects open simultaneously
- Auto-save every 30 seconds (customizable)
- Version history with unlimited undo/redo
- Project notes and metadata
- iCloud sync for multi-device access

**Markers and Sections**:
- Create section markers (intro, verse, chorus, bridge)
- Color-coded markers for visual organization
- Jump to any marker instantly
- Export markers to MIDI files

### Recording

**MIDI Recording**:
- Record from any MIDI controller
- Count-in and metronome
- Quantize during or after recording
- Multi-take recording with loop cycle
- Overdub and replace modes
- MIDI merge recording

**Audio Recording**:
- Record vocals, instruments, or any audio source
- Low-latency monitoring
- Automatic input monitoring
- Multiple audio interface support
- 24-bit/96kHz recording quality
- Record arm multiple tracks simultaneously

**Step Recording**:
- Enter notes one at a time
- Specify duration and velocity
- Great for precise MIDI input
- Works with or without MIDI keyboard

### Editing Tools

**Region Editing**:
- Split, join, and resize regions
- Non-destructive editing
- Region fades and crossfades
- Region quantization
- Region transposition
- Region reverse

**Note Editing**:
- Piano roll editor for MIDI
- Event list for precise editing
- Score editor for notation
- Tablature editor for fretted instruments
- Drum editor for percussion

**Advanced Tools**:
- Brush tool for painting notes
- Scissors tool for splitting
- Glue tool for joining
- Eraser tool for deletion
- Marquee selection for bulk operations

### Transport Controls

**Playback**:
- Play, pause, stop, and record
- Cycle mode for looped playback
- Punch in/out for recording sections
- Skip forward/backward by measure
- Jump to start/end or any position

**Tempo and Time**:
- Tap tempo feature
- Tempo track for tempo changes
- Time signature changes
- Rubato tempo (free timing)
- Count-in options (1-4 bars)

**Metronome**:
- Customizable click sound
- Accent first beat
- Volume control
- Enable/disable during recording or playback

## Automation

### Automation Modes

**Read Mode**:
- Playback existing automation
- No recording
- Default mode

**Touch Mode**:
- Write automation only while touching control
- Returns to previous value when released
- Great for mixing automation

**Latch Mode**:
- Write automation when control moves
- Remains at last value when released
- Good for volume/pan rides

**Write Mode**:
- Overwrites all automation
- Continuous recording
- Use with caution

### Automation Editing

**Drawing Automation**:
- Pencil tool for drawing curves
- Line tool for straight segments
- Shape tools (sine, triangle, square)
