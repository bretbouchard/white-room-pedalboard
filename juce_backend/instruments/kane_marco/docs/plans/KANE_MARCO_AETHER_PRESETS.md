## Overview

Kane Marco Aether's factory presets showcase the unique physical modeling architecture: **Exciter → Resonator → Feedback**. Each preset is carefully crafted to demonstrate organic, living textures that evolve over time.

## Architecture

### Exciter Section
- **Noise Color:** White (0.0) to Pink (1.0) noise
- **Gain:** Excitation strength
- **Envelope:** Attack, Decay, Sustain, Release shaping

### Resonator Section
- **Mode Count:** Number of resonant modes (4-64)
- **Brightness:** Frequency distribution tilt
- **Decay:** How quickly modes resonate

### Feedback Section
- **Amount:** Feedback level (0.0-1.0)
- **Delay Time:** Feedback delay (1-200ms)
- **Saturation:** Distortion amount (1.0-10.0)
- **Mix:** Wet/dry balance (0.0-1.0)

### Filter Section
- **Cutoff:** Filter frequency position
- **Resonance:** Filter resonance peak

### Amp Envelope
- **Attack, Decay, Sustain, Release:** Output amplitude shaping

## Tips for Best Results

### Ambient Presets
- Use slow attack (0.5s+) for smooth fades
- Layer multiple presets for richer textures
- Apply reverb after Kane Marco Aether for depth

### Cinematic Presets
- Automate filter cutoff for sweeps
- Use modulation on feedback amount for movement
- Layer with traditional orchestral sounds

### Texture Presets
- Trigger with short MIDI notes for percussive effects
- Use velocity to control exciter gain
- Combine with convolution reverb for realism

### Drone Presets
- Hold notes continuously without envelopes
- Use EQ to shape frequency content
- Layer multiple drone presets for richness

### Bell Presets
- Use short, decisive MIDI notes
- Apply high-pass filter to clean up sub
- Layer with real bell samples for impact

### Pad Presets
- Use slow attack for smooth chord changes
- Moderate sustain for natural decay
- Apply stereo widening after the effect

## Technical Notes

### Physical Modeling
Kane Marco Aether uses a physical modeling approach based on modal synthesis:
1. **Exciter:** Generates energy (noise burst)
2. **Resonator:** Filters energy through resonant modes
3. **Feedback:** Recirculates output for complexity
4. **Filter:** Shapes final frequency response

### Mode Distribution
Different presets use different mode distributions:
- **Harmonic:** Integer multiples (bell, pad)
- **Inharmonic:** Prime number ratios (sci-fi, tension)
- **Natural:** Logarithmic distribution (texture, organic)

### Saturation Curve
Feedback saturation uses a soft-clipping curve:
- 1.0-2.0: Warm saturation
- 2.0-5.0: Rich harmonics
- 5.0-10.0: Aggressive distortion
