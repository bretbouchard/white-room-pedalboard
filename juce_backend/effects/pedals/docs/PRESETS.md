# Pedals Presets

## Factory Presets

Each pedal includes a collection of factory presets showcasing its capabilities.

## Preset Locations

Factory presets are stored in `presets/factory/` and are loaded automatically by the plugin.

## Preset Formats

- **VST3**: `.vst3preset` files
- **AU**: `.aupreset` files
- **CLAP**: Presets embedded in plugin
- **Standalone**: XML-based preset files

## Creating Custom Presets

Users can create and save custom presets via the host's preset save functionality.

## Preset Sharing

Presets can be shared between users by copying the preset files from the host's preset directory.
